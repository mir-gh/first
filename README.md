# Titre alternatif

## Sous-titre alternatif

Les paragraphes sont séparés
par une ligne laissée vide

Deux espaces à la fin d'une ligne produisent  
un saut de ligne.

Les caractères _italiques_, **gras**, `à taille fixe`.

  1. fruits
     * pomme
     * banane

     - carotte
     - brocoli

Liste à cocher:
 - [ ] Case non cochée
 - [x] Case cochée
